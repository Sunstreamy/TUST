meta、h3、hr等标记的应用

HTML5(英文全称：HyperText Markup Language 5)是构建Web内容的一种语言描述方式。它是互联网的核心技术语言，由万维网联盟（W3C：World Wide Web Hypertext Application Technology Working Group，简称：WHATWG）创建。HTML 标准自 1999 年 12 月发布的 HTML 4.01 后，后继的 HTML5 和其他标准被束之高阁，为了推动 Web 标准化运动的发展，一些公司联合起来，成立了一个叫作 Web Hypertext Application Technology Working Group（Web 超文本应用技术工作组，WHATWG）的组织。
